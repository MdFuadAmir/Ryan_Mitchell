# 🎨 Graphics Designer Portfolio Website

A modern, responsive and premium portfolio website built for a professional graphics designer.  
This project showcases creative works, services, pricing plans, and contact functionality with email integration.

---

## 🌟 Project Overview

This is a **fully responsive single-page portfolio website** designed for a freelance graphics designer named **Ryan Mitchell**.  
The goal of this project is to present skills, services, and creative works in a visually appealing and conversion-focused layout.

It is built with modern UI principles including soft gradients, smooth animations, and interactive components.
